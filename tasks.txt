1. Проходит по всем постам тг канала
2. Заносит в БД информацию о постах: id поста, текст (включая форматирование), медиа. Форматирование вставляется дополнительными тэгами маркдауна, то есть если текст жирный, то добавляет ему "*" до после текста. ТГ сам тебе с ними отдавать должен вообще
3. Раз в 30 минут делаешь проверку наличия новых постов. В случае добавления нового поста, у него новый последовательный id, поэтому ты начинать с последнего поста в БД и идти дальше. Если возвращается null, то значит, поста нового нет, иначе добавляешь новую запись в БД
4. API отдача записей по запросу
